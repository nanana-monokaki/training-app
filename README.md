# Roadbike Training App

スマホで使うロードバイク向けトレーニングメニューアプリです。

## Features

- ツールド東北180km、短期ロング、通常期のメニュー切替
- FTP154W基準のZwift強度表示
- 曜日別の開閉カード
- 筋トレ種目のYouTube埋め込み
- 体重、体脂肪率、疲労度、状態、メモの端末内保存
- URLパラメータ `?weight=57.2&bodyFat=21.0` 取り込み

## Health Data

OMRON connectはAppleヘルスケアへデータ共有できますが、通常のWebアプリからHealthKitを直接読むことはできません。
このアプリでは手入力とURLパラメータ取り込みを暫定対応とします。
